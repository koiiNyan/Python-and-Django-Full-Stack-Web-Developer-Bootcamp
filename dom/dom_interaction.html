Grabbing specific elems from the DOM

  • document.getElementById() (Usually it's a single element)
  • document.getElementByClassName() (.class)
  • document.getElementByTagName() ()
  • document.querySelector() - returns the first obj matching the CSS style selector
  • document.querySelectorAll() - returns a list of all matching selectors
